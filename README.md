# 🎸 Air Guitar – Gesture Controlled Virtual Guitar

An interactive **Computer Vision project** that allows users to simulate playing a guitar using **hand gestures detected through a webcam**.

This project uses **Python, OpenCV, and MediaPipe** to track hand movements in real time and convert them into virtual guitar interactions. It demonstrates how **AI and gesture recognition** can create engaging and interactive applications.

---

## 🚀 Demo

Play a virtual guitar using only your **hand movements in front of a webcam**.

Example workflow:
1. Start the program.
2. Show your hand in front of the camera.
3. Move fingers across the virtual guitar area.
4. The system detects gestures and triggers guitar sounds.

## ⚙️ How It Works

1. The webcam captures live video input.
2. **MediaPipe** detects hand landmarks and tracks finger positions.
3. The system identifies gestures based on finger movement and position.
4. When a gesture is detected, it triggers a **virtual guitar interaction**.
